~~~Intro
Этот проект представляет собой бота для техподдержки, использующего искусственный интеллект для автоматической генерации ответов на запросы пользователей. 
Он интегрируется с CRM Pyrus и отвечает на комментарии, поступающие в задачу.
~~~
Для работы проекта необходимо установить следующие библиотеки:

~~~
- quart
- asyncio
- gunicorn
- uvicorn
- python-dotenv
- openai
- mysql-connector-python>=8.3
- bcrypt
- apscheduler
- requests
- aiohttp
~~~

Установите их с помощью команды:

~~~bash
pip install -r requirements.txt
~~~

Проект предназначен для хостинга на Railway. После загрузки кода на Railway, приложение автоматически запустится и будет работать.
1. Убедитесь, что у вас есть учетная запись на [Railway](https://railway.com/).
2. Загрузите репозиторий на Railway с помощью команд:

~~~bash
git init
git add .
git commit -m "Initial commit"
git push origin main
~~~

3. Настройте базу данных в переменных окружения на Railway.

Описание функционала:

1. Бот обрабатывает входящие сообщения и вложения в виде комментариев к задачам.
2. Бот генерирует ответы на основе комментариев с помощью модели OpenAI.
3. Если ответ бота начинается с двух точек (`..`), бот передаст заявку сотруднику техподдержки.
4. В случае некоторых ключевых фраз (например, благодарности (`Спасибо`) или просьба (`AnyDesk`)), бот передаст заявку сотруднику техподдержки.
5. После завершения работы в задаче бот заполняет поля, связанные с задачей информацией, полученной из диалога

Необходимо настроить файл `.env` с ключами API:

~~~
MYSQL_URL
~~~

- Этот бот предназначен исключительно для личного использования.
- Используемая модель OpenAI настроена на основе специфического обучения ([Fine-Tuned GPT-4](https://platform.openai.com/docs/guides/fine-tuning)).
